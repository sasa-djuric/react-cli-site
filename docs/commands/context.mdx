# Context

<Description>Command for creating context file</Description>

```bash
npx react context <name> [options]
```

```bash
npx react ctx <name> [options]
```

## Arguments

| Argument             | Description         |
| -------------------- | ------------------- |
| <code><name\></code> | Name of the context |

## Options

| Option                             | Alias              | Description                                                                                                                    |
| ---------------------------------- | ------------------ | ------------------------------------------------------------------------------------------------------------------------------ |
| <code>--typescript</code>          | <code>-t</code>    | Use typescript                                                                                                                 |
| <code>--customProvider</code>      | <code>-cp</code>   | Use custom provider                                                                                                            |
| <code>--hook</code>                | <code>-h</code>    | Create a hook for the usage of context                                                                                         |
| <code>--no-typescript</code>       | <code>-notp</code> | Don't use typescript                                                                                                           |
| <code>--no-customProvider</code>   | <code>-nocp</code> | Don't use custom provider                                                                                                      |
| <code>--no-hook</code>             | <code>-noh</code>  | Don't create a hook for the usage of context                                                                                   |
| <code>--path <destination\></code> | <EmptyColumn />    | Specify the path for the file relative to the project source path defined in the [project configuration](/docs/config/project) |
