# Component

<Description>Command for creating component files</Description>

```bash
npx react component [type] <name> [options]
```

## Arguments

| Argument             | Description                                                                                                                                                 |
| -------------------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------- |
| <code>[type]</code>  | Type of component defined in the configuration file. If it's not provided, a default will be used                                                           |
| <code><name\></code> | Name of the component. If the forward-slash is used in the name, everything before the last slash will be treated as an addition to the component type path |

## Options

| Option                             | Alias              | Description                                                                                                                     |
| ---------------------------------- | ------------------ | ------------------------------------------------------------------------------------------------------------------------------- |
| <code>--class</code>               | <code>-c</code>    | Create a class component                                                                                                        |
| <code>--typescript</code>          | <code>-tp</code>   | Use typescript                                                                                                                  |
| <code>--style</code>               | <code>-s</code>    | Create a style file                                                                                                             |
| <code>--story</code>               | <EmptyColumn />    | Create a [story](https://storybook.js.org/docs/react/get-started/introduction) file                                             |
| <code>--proptypes</code>           | <code>-p</code>    | Use proptypes                                                                                                                   |
| <code>--redux</code>               | <code>-r</code>    | Use redux                                                                                                                       |
| <code>--test</code>                | <code>-t</code>    | Create a test file                                                                                                              |
| <code>--index</code>               | <code>-i</code>    | Create an index file with export declaration(s)                                                                                 |
| <code>--inFolder</code>            | <code>-f</code>    | Create a component in its own folder                                                                                            |
| <code>--file-name <name\></code>   | <code>-fn</code>   | Specify the file name. If this argument is provided, the [file naming config](/docs/config/project#file-naming) will be ignored |
| <code>--path <destination\></code> | <EmptyColumn />    | Specify the path for the file relative to the project source path defined in the [project configuration](/docs/config/project)  |
| <code>--no-typescript</code>       | <code>-notp</code> | Don't use typescript                                                                                                            |
| <code>--no-style</code>            | <code>-nos</code>  | Don't create a style file                                                                                                       |
| <code>--no-story</code>            | <EmptyColumn />    | Don't create story book file                                                                                                    |
| <code>--no-proptypes</code>        | <code>-nop</code>  | Don't use proptypes                                                                                                             |
| <code>--no-redux</code>            | <code>-nor</code>  | Don't use redux                                                                                                                 |
| <code>--no-test</code>             | <code>-not</code>  | Don't create a test file                                                                                                        |
| <code>--no-index</code>            | <code>-noi</code>  | Don't create an index file or add declarations in the existing one                                                              |
| <code>--no-inFolder</code>         | <code>-nof</code>  | Don't create a component in its own folder                                                                                      |
