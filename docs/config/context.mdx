# Context config

<Description>Context configuration object</Description>

| Option                      | Type                        | Description                                                                                                  |
| --------------------------- | --------------------------- | ------------------------------------------------------------------------------------------------------------ |
| <code>open</code>           | <code>boolean</code>        | Open file in default editor after creating                                                                   |
| <code>path</code>           | <code>string</code>         | Specify the path for the file relative to the project source path defined in the project configuration entry |
| <code>typescript</code>     | <code>boolean</code>        | Use typescript                                                                                               |
| <code>inFolder</code>       | <code>boolean</code>        | Save context file in its own folder                                                                          |
| <code>customProvider</code> | <code>boolean</code>        | Create custom wrapper component around context provider                                                      |
| <code>hook</code>           | <code>boolean</code>        | Create hook inside context file for context usage                                                            |
| <code>fileNaming</code>     | [File naming](#file-naming) | Specify file naming preferences                                                                              |
| <code>export</code>         | [Export](#export)           | Specify export preferences                                                                                   |

## File naming

| Option              | Type                                                                                                 | Description                                                                           |
| ------------------- | ---------------------------------------------------------------------------------------------------- | ------------------------------------------------------------------------------------- |
| <code>name</code>   | <code>string</code>                                                                                  | Specify name. [Name placeholders](#name-placeholders) are available for usage in name |
| <code>casing</code> | <Space><code>camel</code> \| <code>pascal</code> \| <code>kebab</code> \| <code>snake</code></Space> | Specify casing type                                                                   |

### Name placeholders

| Type                | Description   |
| ------------------- | ------------- |
| <code>{name}</code> | Provided name |

## Export

| Option                   | Type                 | Description                                                   |
| ------------------------ | -------------------- | ------------------------------------------------------------- |
| <code>default</code>     | <code>boolean</code> | Specify export type (default or named)                        |
| <code>inline</code>      | <code>boolean</code> | Specify should export be declared inline                      |
| <code>destructure</code> | <code>boolean</code> | Destructure Provider and Consumer components from the context |
