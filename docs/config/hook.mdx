# Hook config

<Description>Hook configuration object</Description>

| Option                  | Type                                            | Description                                                                                                  |
| ----------------------- | ----------------------------------------------- | ------------------------------------------------------------------------------------------------------------ |
| <code>open</code>       | <code>boolean</code>                            | Open file in default editor after creating                                                                   |
| <code>path</code>       | <code>string</code>                             | Specify the path for the file relative to the project source path defined in the project configuration entry |
| <code>typescript</code> | <code>boolean</code>                            | Use typescript                                                                                               |
| <code>inFolder</code>   | <code>boolean</code>                            | Save the hook file in its own folder                                                                         |
| <code>fileNaming</code> | [File naming](/docs/config/project#file-naming) | Specify file naming preferences                                                                              |
| <code>export</code>     | [Export](/docs/config/project#export)           | Specify export preferences                                                                                   |
