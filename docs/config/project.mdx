# Project config

<Description>Project cofiguration object</Description>

| Option                  | Type                                   | Description                                        |
| ----------------------- | -------------------------------------- | -------------------------------------------------- |
| <code>path</code>       | <code>string</code>                    | Project source path                                |
| <code>typescript</code> | <code>boolean</code>                   | Uses typescript                                    |
| <code>lint</code>       | <code>boolean</code>                   | Uses eslint if it's used to lint created files     |
| <code>format</code>     | <code>boolean</code>                   | Uses prettier if it's used to format created files |
| <code>verbose</code>    | <code>boolean</code>                   | Outputs operation and file path on stdout          |
| <code>fileNaming</code> | <code>[FileNaming](#filenaming)</code> | Define file naming                                 |
| <code>export</code>     | <code>[Export](#export)</code>         | Define export preferences                          |
