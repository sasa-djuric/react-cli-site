# Style config

<Description>Style configuration object</Description>

| Option                  | Type                                                                                                                             | Description                                                                                                  |
| ----------------------- | -------------------------------------------------------------------------------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------ |
| <code>type</code>       | <Space><code>css</code> \| <code>scss</code> \| <code>sass</code> \| <code>less</code> \| <code>styled-components</code></Space> | Type of styling technology                                                                                   |
| <code>modules</code>    | <code>boolean</code>                                                                                                             | Use css modules                                                                                              |
| <code>path</code>       | <code>string</code>                                                                                                              | Specify the path for the file relative to the project source path defined in the project configuration entry |
| <code>typescript</code> | <code>boolean</code>                                                                                                             | Use typescript (for css in js solutions)                                                                     |
| <code>inFolder</code>   | <code>boolean</code>                                                                                                             | Save the file in its own folder                                                                              |
| <code>fileNaming</code> | [File naming](/docs/config/project#file-naming)                                                                                  | Specify file naming preferences                                                                              |
| <code>export</code>     | [Export](/docs/config/project#export)                                                                                            | Specify export preferences for CSS in JS styling solutions                                                   |
